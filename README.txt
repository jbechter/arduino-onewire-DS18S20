DS18S20 Arduino OneWire Library

To install, download and rename the arduino-onewire-DS18S20 folder to DS18S20 and copy the DS18S20 folder structure to your Arduino libraries folder.

Requires Arduino 1.0 or greater and OneWire Arduino library (see http://playground.arduino.cc/Learning/OneWire).

For additional information see http://projects.bechter.com/arduino-onewire/ds18s20
